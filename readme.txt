Starts a program after a specified number of seconds.

Author: Hallvard Nygard, hn@jaermuseet.no
Location: http://github.com/hnJaermuseet/waitAndRun

Example of use:
- Make a shortcut for the waitAndRun.jar
- Add number of seconds and the program that needs to be excecuted after the (...)waitAndRun.jar"
- Put the shortcut in the startupfolder in Windows

"C:\path\to\waitAndRun\waitAndRun.jar" 5 "C:\Program Files\Mozilla Firefox\firefox.exe"
